# JACK Landing Page

> Your City, Connected.

**Live Site:** https://praveenkg02.github.io/jack-landing/

## About

Landing page for JACK — a local commerce platform to book services, discover local vendors, and earn rewards.

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Animations)
- Vanilla JavaScript (Intersection Observer API)
- Google Fonts (Space Grotesk, Space Mono)

## Features

- Responsive design
- Scroll-triggered animations
- Phone mockup UI
- Interactive map visualization
- Loyalty rewards showcase
